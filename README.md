# 🚀 PulseRun (펄스런)
실시간 시장의 맥박(Pulse)을 감지하고, 기회를 향해 달리는(Run) 스마트 알람 플랫폼

# 📝 프로젝트 소개
PulseRun은 요동치는 암호화폐 시장의 데이터를 실시간으로 트래킹하여, 사용자가 설정한 전략에 맞춰 즉각적인 인사이트를 제공하는 지능형 알람 서비스입니다. 단순히 가격만 알려주는 서비스를 넘어, 시장의 급변하는 흐름을 놓치지 않고 대응할 수 있도록 돕습니다.

# ✨ 핵심 가치 (Why PulseRun?)
Zero-Lag Realtime Sync: 업비트 등 주요 거래소 API를 활용해 지연 없는 실시간 시세 및 거래 체결 데이터를 수집합니다.

Custom Strategy Alert: 단순 지정가 알람은 기본, 변동성 돌파나 기술적 지표(RSI, MACD 등) 기반의 고도화된 알람 설정을 지원합니다.

Multi-Channel Delivery: 텔레그램, 디스코드 등 사용자가 원하는 채널로 24시간 끊김 없는 알림을 발송합니다.

Expandability: 향후 모의 투자 및 자동 매매 엔진으로 확장 가능한 모듈형 구조로 설계되었습니다
